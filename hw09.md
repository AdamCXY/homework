# 牧师与恶魔过河

1.  |类别      |          |
    | -------- | -----:   |
    |           |角色     |
    |           |环境     |
    |           |工具     |

2. 角色：恶魔（3个）；牧师（3个）；

    环境：河流（一条）；天空（1片）；岸（两岸）；

    工具：小船（一艘）；座位（两个）；

3. 类是对象的抽象；对象是类的实例。比如：水果是类，而苹果是对象。

4. |属性      | 方法                 |
   |-------   | -----:              |
   |杀死牧师   |人数大于牧师时杀死牧师 |
   |过河      |坐在船上过河          |

5. Procedure 

   Onclick

     If(恶魔在岸上）

        Then（恶魔翻跟头进入船中）

        else（恶魔从船上翻跟头到达岸上）
